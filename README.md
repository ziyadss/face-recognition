# face-recognition

A facial recognition system.

## Prerequisites

- Python 3
- pip

## Installation

To install the required libraries, run the following command:

```bash
pip install -r requirements.txt
```

## Usage

### Detection

```bash
python -m detection.train   # train the model
python -m detection.main    # run the demo
```
